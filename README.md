# MusiBoard

Basic Scoreboard management plugin for Paper.

Groovy:

```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.Musician101:MusiBoard:[version]'
}
```

Kotlin DSL

```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    api("com.github.Musician101:MusiBoard:[version]")
}
```
